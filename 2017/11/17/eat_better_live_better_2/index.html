生活不仅是写代码，还有吃喝玩乐-谈谈天山SOHO附近的吃

发表于 2017-11-17   |     |   阅读次数

总不能天天坐在电脑前，毕竟附近还有一大堆新事物等着去挖掘。因为我们人数比较多，每顿饭又经常玩游戏，所以选择的地方都是有大桌子并且预算在¥30左右的店。前段时间说过了易园附近的吃，最近天山SOHO附近的羊毛薅的也差不多了，跟大家分享一下。

地下是四通八达，附近几个十字路口都可以通过地下通道走到。而从办公室出去主要有两条线路，，一条是直接去南丰城，一条是出门往娄山关路地铁站那边走。

花开并蒂，各表一枝，先说南丰城。往南丰城这边走右手第一家是简厨，对，就是我冲了3k的那家湘菜店，都是长桌，人多的时候都是点两份一样的菜，分两桌吃，坐中间的人可以两边一起吃。不充钱的时候，点的菜数=人数-1；充完钱后可以，点的菜数=人数+1，吃的就比较舒服。饭是按人头算的，菜比较下饭，味道还可以。一般预算35左右。

简厨旁边是天辣，一家连锁店，味道还算可以，小炒肉挺合我口味，每次都有¥5元的优惠券，也是长桌，点的菜数=人数，人均¥35左右。

天辣的对面是谷田稻香，他家是套餐，吃过一两次，主要是因为现在吃不动锅巴了，不是很喜欢去吃。

往前一点是和府捞面，早前大众点评有个酸菜雪花肥牛面优惠券，相当划算，每次吃都要再续一碗面。对，他家免费续面吃到饱，绝对良心。后来肥牛面取消了，改成大师素油拌面，就去的少了。

和府捞面旁边是一家乌冬面，也有套餐，吃了一次，不是很喜欢过去。

继续往前走，先不要急着上楼，快要走到尽头的时候，有一节楚楚馅饼粥，他们家有圆桌，整个地下就他们一家有圆桌。可以点菜，也可以吃馅饼什么的，伟大一般般，主要是圆桌加分。

馅饼粥往前走一点是吃混沌的，那种大混沌我吃不来，还是小混沌比较合我口味，价钱不到¥30。

对面是一家麻辣烫，还有优惠券没用的，麻辣烫味道也就都那样了。

接着就是最后一家了，蒸九香，两种选择，一种是打菜，类似学校食堂那种，赶时间可以吃，还有就是石锅套餐，味道挺好的。

前文有说到可以上楼，楼上店面的消费都比较高，只推荐一家，云彩泥，有个4人¥168套餐，适合5人去吃，蛮划算，味道也还行。他们家也只有长桌可以拼。

南丰城吃过的就是上面几家，都是在地下，想呼吸点新鲜空气的话就去往娄山关路那边走，走地上，下雨的话可以考虑走地下，虽然地下也有吃的，但都是小吃类型，不太适合做正餐。

往娄山关路这边走，走到第一个路口，先看到的是汇金百货，汇金百货一楼有个味千拉面门店，可以用招行APP买优惠券，相当划算，当一个人只能用一次，用完一次后我就再也没去了。

汇金百货里面负二楼有个70后饭吧，菜量很小，吃不饱，味道一般般，主打特色是物品怀旧，换口味的时候会过来。里面也有其他吃的，但吃不起就买去看过。

汇金百货路口斜对面有家百盛，我们通常都是去五楼，首推外婆家吧，一直爆满，每次打电话过来都收没有圆桌，但我觉得味道也就一般般啊。可能是大家都觉得便宜吧。

外婆家旁边有家烤鱼，说实话是我吃过的最烂一家烤鱼，也不知道是不是恰好我吃的那顿不好吃。

还有家潮汕牛肉火锅，虽然有同事跟我说他们过来吃可以控制人均到¥30左右，但是我自己吃过一顿，预算大大超了，没敢再带集体过来吃，不然他们肯定不给钱。

明泰谷，泰国菜，只吃¥37套餐的，吃的很饱，但套餐种类很少，就三样。支付宝可以领一张代金券，但一人只能用一次。

明泰谷对面是云海肴，报公司名字可以打八折，去的次数比较多，喜欢做靠窗的位置，感觉不错。人均¥35左右就ok了。

百盛对面是巴黎春天，地面有家潮汕菜馆，店面很小，人多都是没位置，有次自己来吃过一次，后来就拉黑了，不合口味。

巴黎春天地下有家矢作川，日料，说实话跟中山公园附近的差远了，但这也是我找得到能去吃的起的最近一家日料店了。

日料附件还有家沙利亚，嗯，蛮便宜的，桌子也够长，主要还是便宜。

巴黎春天这边走到头是耶里夏利，非常体现我点菜功力的一家店，也有招行的APP可以买拳用，不点羊肉串的话，预算在¥35左右，点串的话，就逼近¥40了。最主要是它们家桌子足够大，可以容纳得下是十二三人。

百盛后面还有家金虹桥，是在地下，这边就是日本人居多了，都是日式的店铺，包括超市之类的。但是这边的日料店价格不亲民。负一楼的话有各种铁板啊，或者套餐什么的。味道中规中矩吧，吃不出特别的，换口味可以来看看。
