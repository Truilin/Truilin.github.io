入职一周年-谈谈易园附近的吃

发表于 2017-05-30   |     |   阅读次数

去年的儿童节来的公司，一晃一年过去了，秉承着“eat better，live better”，这里想说说易园附近的吃。

易园坐落在安化路，感觉大部分人第一次来的时候都会直接路过，完全看不出这里藏着一家公司。易园有个园区食堂，论味道比学校食堂差远了，菜品又少，价格倒还适中，吃过一两次完全不想再来了。以易园为中心，发散开来，倒是有很多合适作为工作餐地点的选择。

尚肉

尚肉就在公司楼下，如果下雨的话，这家是第一选择，较为精致，免费加饭，每个人点自己的套餐，花费在30元左右，味道也还可以。

美鲜鲜果业

出了公司们往东走，没多远有一家水果店，这家店主要是想推荐下他家的椰子，之前有5元一个的，超赞，后来老板说进不到货了，另外一种6元一个也蛮不错，吃完饭回来顺手可以买一个。

粥店

美鲜鲜果业旁边有个粥店，之前组里有同事拔牙后，经常来这里喝粥，我是觉得不大吃不饱。人均15元左右。

包大面菜馆

继续沿着安化路往东走，这里有一节家常菜馆，之前老板还卖梅干菜扣肉饼，而且老板是开心了就卖几张饼，不开心就不卖，现在已经好久没看到卖饼了，五元一个。里面都是盖浇饭什么的，我是觉得不很合我口味。

馄炖店

包大旁边就是一个很小的馄炖店，馄炖分大小两种，大的就跟饺子一样，吃不来，吃过一次没来过第二次。

牛肉拉面

馄炖店斜对面有家牛肉拉面，基本每次来人都感觉坐满了，要是赶时间或者下雨，会来这边吃个吃个盖浇面盖浇饭什么的，人均十几块钱。

祥和小菜&面馆

安化路走到头是一个十字路口，两边一家是祥和小菜，一家时祥和面馆。他们的logo都是一样，我一直觉得是一个老板开的，虽然从未求证过。祥和面馆从没进去吃过，祥和小菜进去吃过几次，可以点菜，但是感觉这家卫生不怎样。人均不到二十

牛首掌

十字路口这里有家卖饼的，有些同事喜欢吃梅干菜的饼，我觉得牛首掌也不错，有时候没吃饱会来一张饼，五元一个，味道还可以。

杨国福麻辣烫

十字路口是安化路和定西路的交汇，沿着定西路往北走，这里有家杨国福，肠胃不好的人还是少来吃，人均25左右，看自己点菜水准。

大椒小厨房

杨国福的路上就是大椒小厨，里面有两个套餐，一个是烤鱼套餐，一个是牛蛙套餐，然而同事有忌口的，这家也是牛蛙做的多。味道还可以吧，人均30+的样子。

浏阳蒸菜

大椒小厨再往上一层就是新开的浏阳蒸菜，便宜是最大的特点，人均十块钱出头就可以了，可以几个人一起点菜，也可以各点个的。

老盛兴

杨国福对面是老盛兴，说是老字号什么的，其实也就是盖浇饭什么的，而且我觉得不是很好吃，人均二十不到的样子，各点各个的。

百素酸菜鱼

老盛兴上面就是酸菜鱼，味道蛮不错的，但是这家的预算我控制不下来，过来吃顿都是40+，而且周末人都很多，队伍很长。组里也有同事不吃鱼，所以每次他不在我都会考虑这里。味道蛮可以。

韩伊拌饭

老盛兴旁边有个小巷子，进去就是一家韩伊拌饭，味道一般般吧，人均二十来块。

台北101面馆

老盛兴沿着定西路继续往北走，又是一个小箱子，里面是一个面馆，服务员从来都是戴着帽子。虽然我不喜欢这家的面，但是还是来吃过几次。人均三十左右。

新疆亚克西烤羊肉串

定西路还是往北走，这里就有一家羊肉串，强推！5元一串，老板都是当着面串肉烤肉的，而且量还足，真良心！从来都是排队。

摊摊面

羊肉串旁边就是一家小面，过来吃面的时候可以就着旁边的羊肉串。

永和大王

定西路还是向北走，永和大王，普通的快餐，节省时间可以考虑下，人均二十来块。

潮汕牛肉火锅&川西坝子

永和大王过了没几步就是一个三叉路口，直走的时候路边有两家火锅，川西坝子不推荐，潮汕那家到时可以点套餐，味道还蛮不错。

牛丼

三岔路口，东北方位的商场，里面吃的很多，价格也稍高些。进楼第一层就是牛丼，人均25左右，味道还行吧。

厨房乐章

商场二层有个厨房乐章，逼格还可以，每次工作餐我都控制预算在人均30左右，建议点菜数=人数，适合九个人左右一起。

沙利亚

商场四层是沙利亚，还是蛮便宜的，当然也可能是个假意餐，人均30左右。

丰收日

商场五层是丰收日，原价真特么贵，但是工作日有优惠，我还办了个会员卡，有需要的可以找我借。人均也是控制30以内的。

大娘水饺

出了商城，定西路上就有一家大娘水饺，也是只吃过一次不想再来了。

绿茶

大娘水饺对面看上去是个停车厂，但是绕一绕可以绕到绿茶去，人均可以控制在30以内，味道一般般吧，菜量比较少。

银

定西路这边走到头就是一个十字路口了，十字路口往东走，没几步一个很小的楼梯可以上去，是一家日料，日料我去吃的时候都是中午吃套餐，其他时间段吃不去，银这家味道还可以。人均也是30左右就可以了。

振鼎鸡

定西路这条路往北走到头暂时先不走了，往回退到和安化路交汇的十字路口，再沿着安化路往南走，先是碰到振鼎鸡，一家只收现金，一家只做鸡的商家。晚上七点左右过去，基本鸡就卖完了，其实就是普通的白切鸡，但是蘸料味道还可以，人均也是30左右。

黄焖鸡

振鼎鸡对面就是一家黄焖鸡啊，大家基本都吃过，没啥特别的。

重庆鸡公煲巫山烤鱼

黄焖鸡旁边就是一家重庆鸡公煲，这种东西吃饭后，身上残余味道太重了，味道也是一般般吧，人均预算40+。

熊猫串串

贵，贵到我问完价格默默退出没有吃他家的东西。

羊肉汤

串串对面是一家卖羊肉汤的，泡饼吃，但有次在他家吃的时候，一只飞蛾当面飞进了汤里面，再没踏进过一步。

斑鱼庄

定西路往南走就是和武夷路的路口了，十字路口有一家斑鱼庄，吃过一次，给过唯一的一次差评。

小南国烧烤

这也是一家从来路过没有进去过的店，不做评价。

定西路浏阳蒸菜

跨过十字路口，定西路继续往南走，其实武夷路往东往西也有吃的，往西有家日料，往东也有些吃的，但没怎么去过。 这家浏阳蒸菜蛮对我胃口，价格也很便宜，很推荐的一家。

云南菜

蒸菜旁边就是一家云南菜，我总觉得云南菜的原料很稀奇古怪，每次让我点菜都很纠结，从菜名透露的信息很难去点菜。他家中午有快餐，但我不喜欢吃。

东北菜

云南菜紧挨着的就是东北菜，量大份足，点菜数=人数-1，就可以了。味道一般般。

重庆小面

在走几步就是重庆小面，从来都是满座的，而且可以续面，简单点说就是管饱，蛮不错的。

银厨

定西路往南再走又是一个十字路口，昭化路往东走几步，就是银厨。这是一家我闭着眼睛都可以点菜的，因为来的次数太多了，基本每周都会来一次。你点什么菜都可以，只要不听老板娘推荐的就好。

玉桂坊

银厨对面就是玉桂坊，一家家常盖浇饭，一般般，可以试试。昭化路上也有家东北菜，吃过一次，不想再去第二次。

花之舞

刚刚的十字路口，昭化路往西，有家日料，花之舞也还可以，饭后可以选择布丁或者冰淇淋，都蛮不错。

达乐披萨

十字路口，定西路往南几步就是这家披萨，便宜，吃过一次。

烤羊&小龙虾&。。。

你要是还是顺着定西路往前走，从高架下走过，又回发现一片新天地，但是那里太远了，不建议作为工作餐的地方了，而且价格也高，等位的人也多。我自己就来吃过几次，还都是放假过来的。

盛贺美

继续回退，退到公司门口，安化路上，往西，路上基本啥吃的都没有，没事，继续走。走到十字路口，就是凯旋路了，凯旋路往北是个三叉路口，走东北方向的汇川路岔路，在这个路上有家日料，较贵，人均40+，不是很建议作为工作餐。

兆丰美食广场

汇川路往前，走到头是个十字路口。这里有家美食广场，负一楼，啥都有，麻辣烫，米线，浏阳蒸菜，牛肉面，吃的较多，铁板烧。。。。不做一个个的评价，可以一个个试过去，都差不了多少。

龙之梦

刚刚的十字路口可以看到龙之梦商场了，里面地下有很多吃的，楼上也有很多，但作为工作餐能吃的不多。

和府捞面

龙之梦几楼我忘了，可以在大众点评上搜一下，有团购券可以用，还是很优惠，免费续面，管饱的。

薛之谦

这个不是工作餐来吃的，吃过几次都是和同学一起的，虽然我觉得一般般。

泰妃殿

泰国菜，可以点菜，中午过来也偶套餐，人均30+，感觉已经吃多了，写的时候我都感觉吃不下了。

京荟萃

新发现的一家店，套餐很划算的，推荐
