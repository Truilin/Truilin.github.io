人生总是由无数的节点组成-落户小结

发表于 2017-09-11   |     |   阅读次数

留上海，和很多人聊过这个话题，是为了户口么，有什么用？除了北京，杭州、深圳，互联网不见得就比上海差了，上海又买不起房，户口聊胜于无。

准备工作

应届生积分落户是相对来说落户最简单的一条路径，通常上海本地211以上高校，过了四六级，没挂科，就可以躺着落户。但总是有各种各样的幺蛾子，这里稍微写一下硬性条件。
1、上海2017年落户门槛还是72分，这个没有变过，积分细则见2017年非上海生源应届普通高校毕业生进沪就业 申请本市户籍评分办法；
2、上海本地高校说是隐形加2分，这个一直是口口相传的，从未见诸于任何纸面文件；
3、我自己没有去算过分数，也没有额外加分，没有任何奖项和专利，没有挂科，全靠着计算机硕士身份躺拿的；
4、说一下四六级只要有证就好，不用在意在哪考的。后面说要盖章，也不用回报考点去盖章，直接在最高学历处盖章就好。
5、上海西南某高校电院对于成绩，只要没挂科，就是给的一等。
6、三方协议，主要就是有人毁约会有麻烦事，某为善于一个拖字，所以要是想先签个保底的同学也要想好了再签。
7、租赁备案，和房东说好就行了，房东要是不愿意办可以找个有房的本地同学或同事帮忙。有些区办这个还要收费用，5%来着？长宁区不用

如果，你分数达不到的话，后面也没有参考什么意义了。

批复（关于同意非上海生源高校毕业生办理本市户籍的通知）

你分数算算差不多的话，把相关的奖项材料，学历学位等复印件交给公司的hr，这些是他们去帮忙办的。
第一联公司会在办证名信的时候用到，之后公司就不用了，自己要拿到，因为上面写了最后要交给落户派出所；
第二联写的是交给学校，但是我们学校根本就没要，看你分数够了后面的事都给省了；
第三联是公司留存。

报道证

分数达到后，就是回学校领报道证了，不同学校（院）情况不一样，报道证有时候还要档案寄送地址写好后再给开，有些需要之前提到的第二联批复。报道证有效期是90天。

高校毕业生申报户口证明信

公司拿着你提供的报道证复印件、学历学位复印件、第一联批复帮你去办证民信，拿到证明信后，后面的流程就和公司没关系了。用人单位办理2017年非上海生源 《高校毕业生申报户口证明信》须知

户口迁移证、准迁证

记得带好所有的材料，尤其的是户口本！！！户口本复印户主信息页和本人信息页，以及以上的材料都记得带上原件和复印件。
这里开始就是大坑了，对于户口在外地的同学，需要先去区公安分区办理户口准迁证，一共三联，第一联分局自留，第二联给原户籍所在派出所，第三联给迁往派出所。有些区公安局不让办，比如浦东就是这样，让去派出所，但是派出所又一拖再拖，我一个同学拖了一个多月都没办好。长宁区还是效率不错的。
第二联准迁证+户口本+身份证拿到原户籍所在地办理户口迁移证。可以代办，各地派所说要求可能不一致，记得打电话询问。

户籍证明

走到这一步就是说落户成功了，外地户口的同学带着户口迁移证，准迁证、以及以上的材料原件和复印件区所租小区的派出所。户口在学校的同学可能要个户口信息页，这个是要回学校去办的，但也有些派出所是不需要的，自己记得打电话问好。
是不是你到老派出所就可以直接办了？当然不是啊！我就觉得自己办的那天是头猪，跑了一个上午！
先去拍个身份证照，这个照片可以在区公安局拍，但是人家说户口迁移证，所以我当时没发拍，但是也有别的拍照点。拍照这个事情在拿到户口迁移证就可以去拍了。
然后申报户口事项申请表，说是可以在公安局官网下载的。我记得当时我坐在窗口，民警问我申请表呢？我张着嘴一脸懵逼，心里直骂自己是头猪。然后民警自己打印了一张表格让我去填。申请表这个是要社区民警签字的。上海市公安局窗口服务告知单（2017年非上海生源应届普通高校毕业生办理落户）
材料都齐全后，就可以当场办理了，户籍证明的有效期是60天，记得多开几张，因为后面会要的。主要是民警也告诉我，这是第一次开，可以多开几张，省的麻烦，以后每次要开户籍证明，都要先写申请书，找社区民警签字，再来派出所办理。之前找社区民警签字的时候，人家也告诉我，说记得以后换个地方租的时候，把户口也记得牵走，省的每次都去麻烦他，或者自己买个房子，有自己的户口本也省事。（但我也有同学派出所办户籍证明不需要社区民警签字，这我又能说什么呢？）orz
最后收个20元，说一个月后来领新身份证。（也可以申请加急的，一个礼拜就到了，我是生生的等了一个礼拜）

劳动手册

户籍证明有了之后，才可以去办劳动手册。劳动手册不办的话，社保交的还是外地，不会算在上海（工作人员告诉我的）。落户之前，有些同学可能把户口已经迁到上海高校了，还有像我这样的户口一直在老家。相比较而言，前者要省去很多事。劳动手册和社保卡是在社区事务中心办理的，和办理租赁备案是在一个地方的。
单单说一下我办理过程的坑吧，去问过社区服务中心后，要求我把准迁证、迁移证、通知单、证明信、户籍证明都放一份到档案里面去，户籍证明是原件，其余的几样复印件加盖派所出公章，工作人员是告诉我如果不放的话档案会被打回去的（所以所有的材料一定要自己备份好）。等档案寄出去后，先去跟单位办理退工，处于失业状态才给你办理（办理完后再让单位招工），并且还要我的老家户口本有我信息的那一页（两张个人寸照），以及放到档案袋里的材料验证后才给办劳动手册。
我是很麻烦的，要的材料基本是最多的一档，我有同学基本什么材料都没要，直接就给办了劳动手册。

户口

是不是所有步骤都走完了，就高枕无忧了？图样图森破，如果你没买房的话，一直租房，那么户口是跟着你实际居住的地方走的，而且要在当前居住地办理租赁备案，把户口迁到居住地派出所辖区。假如不办会怎样？有的派出所会查你的租赁备案是不是到期了，如果到期了，就不会给你开户籍证明，相当于户口被冻结了。所以最好找亲朋好友或者同事签个长点的租赁合同或者寄宿证明，把户口迁过去，省的老是为租赁备案烦。（租赁备案一定要在70年产权证的房子才可以办理的，酒店式公寓这种是不能办的）

公租房

再多嘴说几句公租房的事情，上海公租房是分区筹房和市筹两种的，市筹是拥有上海户口或居住证都可以去办理的，要求跟本市企业签订劳动合同。区筹的区别在于要跟本区注册地的企业才可以办理（我公司有好几个主体，我签的主体注册地是徐汇区，问了公司是可以更改公司主体，但是要好几个高层同意，并且这样对原来的公司主体算一年内离职，会给下一届落户的造成一定影响，想想还是放弃了长宁区的区筹公租房，感觉长宁区区筹还是很赞的）。至于市筹公租房，打电话问了下说是两年内都没有了，要不就是太他么远了（很多公租房都是被转租了，坑啊，徐汇那边好像是用身份证当门卡，这样就好很多了）。有些同学可能会问，能不能落户到公租房？别想了，能上车的早点上车吧。
