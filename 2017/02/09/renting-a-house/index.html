租房小结

发表于 2017-02-09   |     |   阅读次数

看了二十来天的房子，最后租了一个不是特别满意的房子，预算经过两个大佬室友，不停地上涨……

想想还是学校宿舍好啊，不得不夸一下交大的良心宿舍，空调热水一应俱全，治安，楼管都是相当棒，可惜出了这个门就没这个福了。在这里总结一下最近的租房经验，给各位一个参考。

对于一般的只是租房，自己找二房东也好，中介也好，人才公寓也好，都是可以的，但是应届生打算落户公共社区或者办居住证的话，必须要办一个东西就是租赁备案了。如何优雅地在上海办房屋租赁合同登记备案?），当然最好是找同学或者亲戚办这个事情，还省得后面换房子又要重新办。

房源

1、大的中介，链家，爱屋及乌，这些中介是比较大的，房源也相对多一些，但是是要中介费的，上海中介费是统一的，房东租客两边各收一个月的35%。
2、一房东的来源，豆瓣上鱼龙混杂，很多都是中介的伪装；zuber，好像基本是一房东，只发过贴，但房子都不满意，还有很多是招合租室友的。58和房天下虽然有个人房源，但是也不是很靠谱，基本就是碰运气才能碰到一房东。
3、链家的房源，一部分是在网上放出来了，还有一部分是在内部系统看得到的，这个时候你可以先看中一块区域的房子，再在网上联系中介，跟他说好你的需求，他能找到一些你看不到的房子。

中介的一些套路

1、几套房子，先带你看差的，再去看好的，有一个心理落差，基本后面这套就可以定下来了。
2、逼签，中介恨不得你看完房子直接就带你去签，唬你很抢手（实际上好房子的确很抢手）。
3、租金，链家的中介自己都说被链家炒高了，房东挂牌的时候，他们都给出一个参考价，所以他们家的价钱一般都可以谈，谈多谈少看缘分。
4、一些小中介，他们在网上挂出来的房子要么价格超低，要么图片看起来特好，这种房子你是租不到的。

注意事项

1、无论一房东还是中介还是二房东，大家都要看到房产证再最后去租，给钱。
2、图片天花乱坠，链家的人跟我说这是他们专门摄影师拍的并且有优化，一定要实地考察下，如果和他人合租，最好两个人一起去看房子，不然分开看，最后会有分歧的。
3、中介之间的房源是共通的，一家能带你看的房子，另一家也能带你看。
4、中介可以带你私签，一个中介跟我说，冲销量跟他签，便宜1000块。但是这样有风险，如果找中介的话，不建议大家再动这些小心思了，当然如果你撇开中介直接找房东也行。
