五省九天十夜-我一个人的毕业旅行

发表于 2017-03-27   |     |   阅读次数

总有人去你想去的地方，总有人去见你想见的人————桃太郎

临近毕业，无心工作，毕业典礼还有一段时间，离正式入职还有三个礼拜的空闲，于是开始筹划起了自己的毕业旅行，实习攒的钱，平时不去闯关东省下来的钱，够我考虑一次不在乎预算的旅行。

北起北京，南到三亚，东临上海，西至成都重庆，安徽、浙江、江苏也都逛过，可以说走遍了半个中国，这一次也是要考虑去一下之前没去过的地方，所以同学喊去九寨沟、泰国的时候，和之前去过的地方有些重合，被我略过了；再有就是西藏怕高反，新疆这个季节不够好，青海湖没开，贵州玩不了太久，云南去的人太多，而且又没和我一起的，斯里兰卡找不到足够的人。最后打算去弥补下本科的遗憾，去本科那会就应该去过的地方玩，凤凰-张家界-桂林。

张家界

其实一开始也是考虑了云南，本来周六的票又便宜，但是自己加了一个双休日的班，时间少了两天，又没了特价机票，才退而求其次的。周一看了一天的机票和景点，看去张家界机票还可以，直接买了周二中午飞，下午到的机票，晚上住在张家界森林公园山下，第二天上山。

离开上海的时候，天气还是特别好，悲剧的是到了张家界天气就阴了，到的当天天气还只是阴，如果上山还能看到风景，第二天就悲剧了，上了山全是雾，走在路上都看不到人，还不如在山下转转。

张家界的猴子很野，也不怕人，路上直接抢行人食物，吓得我前面两个人不敢走过去，还是我带过去，我自己也是把手机攥的紧紧的，生怕被抢了。

在森林公园由于大雾的关系，蛮失望，阿凡达那些景色拍摄地也都什么都看不到。在张家界的最后一天下午上了天门山，号称世界上最长的索道，名不虚传，做了几十分钟。看缆车的盘山公路，很有感觉。

天门山上有玻璃栈道，但是我走在上面，没有特别的感觉，在东方明珠的玻璃观光厅我还不敢乱走，在天门山的时候我就大踏步的过去了。

走的太累了，下山的时候时间也比较赶，逛完山顶又是下大雾，又是看不到人，路上只能碰到松鼠，不禁让我又回忆起上次一个人独自摸黑爬黄山的感觉了，本来还在想天梯那么长，3000阶梯肯定没法爬，下来的时候才发现天梯给关闭，没法爬。

总结一下就是天气的原因，对张家界之行并不是很满意，最后的天门山稍微可以，还有大峡谷玻璃桥打算给自己留个念想就没去了。

凤凰古城

当初我的同学带着我的卡片来过，约我来没来，这次也是自己一个人来玩，显然天气还不是太好，看不到什么特别的，附一张从客栈看的图

白天景色平平无奇



坐船的时候，船师傅也跟我说现在景色不是特别好，水量也不大

到了晚上，华灯初上，倒还值得一看



桂林

天青色等烟雨，而我在等你。

桂林是烟雨时分去最好的，市内没什么特别的，象山公园由于举办了春节晚会的分会场，票价涨了一倍，但是景色也没特别的。两江四湖也是要坐船夜游才好玩。

阳朔倒是个很不错的地方

九马画山，“看马郎，看马郎，问你神马几多双？看出七匹中榜眼，能看九匹状元郎。”

带着人名币去旅行，二十元人名币取景地

遇龙河无论漂流还是骑车绕行，感觉都不错

广州

玩山水的时候，只有一个人逛，到了大城市才能找到同学，同学看我是一个人逛可怜，请了一天假带我刷了一天的广州的红色地标。
可能是晚上登塔的原因，并没有很多人

然后去逛了越秀公园，看了广州的“羊城”由来，在孙中山纪念馆里睡了一会。本来同学还推荐爬山或者去看花，感觉实在不符合我的风格，给拒绝了。

第一次到广州，还因为气候过敏，居然发生了荨麻疹，吓哭了，不知道是不是对自己白天嘴碎的惩罚（越往南，越有一览众山小的感觉）

深圳

由于港澳通行的过期，只能望洋兴叹，和深圳的小伙伴约了一顿饭就没有然后了，住了一晚，顺便去诊所打了一针开了点药，身上的包给消下去了。

厦门

去不了港澳，又还有两天的时间，就多逛了下厦门，夸一下厦门的小伙伴，带我装逼带我飞，最后吃完饭送我到机场，广州的几个人只会装死，人走了才跟我说话。

除了这句诗，我实在想不到来扬州的理由是什么。
